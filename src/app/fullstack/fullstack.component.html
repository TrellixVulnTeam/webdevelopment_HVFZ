FullStack Development

FullStack Development is a process where a Person or Developer has an ability to develop both client and server side of an Web Application. As shown in picture on the left side of the page it has an burger image along with names of different web technologies mentioned on it describes that as burger has different kind of veges ,cheese..etc in it . So, In the same way full stack developer has knowledge on
  • Writing code using html , css .
  • Program a browser (like using JavaScript, jQuery, Angular, or Vue)
  • Program a server (like using PHP, ASP, Python, or Node)
  • Program a database (like using SQL, SQLite, or MongoDB)

So, as mentioned above person or developer who has knowledge of these skills such as designing of website , performing backend operations such as connecting and managing database is called as fullstack developer.
